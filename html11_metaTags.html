~~ Meta Tag ~~

Meta Tag (self-closing tag) = data about data. We can tell a web browser our website information.


Website information include:

  • Brief Description = tells the website the description of our website.
  • Keywords = useful for search engine operations.
  • Author = For credits.
  • Viewport = to make the website scalable and cleanly viewable in any devices; device-friendly website.
  • Refresh Rate = refreshes the website in a specific time.
